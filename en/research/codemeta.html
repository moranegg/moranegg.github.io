The Software Heritage archive preserves software from different locations with, what we call, intrinsic metadata. Today we index a few formats of intrinsic metadata files, using CodeMeta - detailed on this blog post .

We also ask from researchers to add codemeta.json files to their repository as a good practice for keeping metadata (see SWH research software guidelines ).
Moreover, we promote the codemeta.json format to be a standard output format for software registries. This is why CodeMeta is really important in the research software landscape and we would like to develop and improve the tooling and the vocabulary with the community around it.
